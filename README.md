## Local development

Prerequisties
* git
* golang
* hugo

```bash
# clone the git repository

# cd in the project directory
cd exampleSite

# Start local dev server
hugo server --themesDir ../..
```
Or Check out [Full Documentation](https://docs.gethugothemes.com/Infinity/?ref=github).

- Onepage template
- Google analytics  support
